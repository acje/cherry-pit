# GND Secondary-Reference Audit

Conservative audit proposing additions to `References:` lines for ADRs that
substantively engage with one or more GND directives. Only ADRs whose
Decision/Rules invoke a GND substrate (knowledge/effects gap, intent vs
mechanism, two-levels-up scope, deviation-with-report, observation mechanism,
backbriefing, lifecycle hygiene, schwerpunkt) qualify — not passing mentions.

GND directive shorthand:

- **GND-0001** name knowledge / alignment / effects gap in Context
- **GND-0002** directives state intent + boundaries, not mechanism
- **GND-0003** two-levels-up context scope
- **GND-0004** deviation permitted but reported
- **GND-0005** every directive names an observation mechanism
- **GND-0006** backbriefing before non-trivial action
- **GND-0007** lifecycle / supersession hygiene
- **GND-0008** schwerpunkt / focus of effort

ADRs already re-parented to GND as primary (excluded from this audit):
COM-0008→GND-0006, COM-0019→GND-0005, COM-0026→GND-0002, COM-0034→GND-0007,
AFM-0015→GND-0003.
